# Heroes APP

## Levantar desarrollo

1. Clonar el repositorio
2. Editar el archivo `.env` con las variables de entorno basado en el archivo `.env.template`
3. Ejecutar `npm install`
4. Ejecutar `npm run dev`

## Modo oscuro 🌓

La aplicación soporta un tema claro/oscuro. Haz clic en el icono de la esquina superior derecha (sol/luna) para alternar entre modos.

El tema se guarda en `localStorage` y también respeta la preferencia del sistema operativo al cargar la página.
